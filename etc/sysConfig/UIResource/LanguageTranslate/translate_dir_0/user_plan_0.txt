《用户体验改进计划》

    深圳市创想三维科技股份有限公司（以下简称“Creality”）提供Creality产品和服务（以下简称“Creality服务”）给用户（以下简称“用户”或“您”）。如您对本《用户体验改进计划》有任何疑问，请在阅读完本文后联系 cs@creality.com。
加入用户体验改进计划允许Creality收集相关信息以分析产品的使用情况，用以改善产品服务和提升用户体验。机器将访问您的产品使用相关信息，例如摄像头的错误识别历史信息和机器日志信息，其中可能包括《隐私政策》中描述的信息。
用户可在机器屏幕的帮助选项中自由选择加入或退出；选择加入即表示用户选择参与用户体验改进计划；如果选择不加入，即可退出用户体验改进计划。
    1. 信息的收集
为了向您提供更好的产品或服务，经您授权参与本计划，Creality将向用户采集基于以下目的的数据
产品诊断
    • 设备信息，包括设备Mac地址、SN号、DN号、国家/地区、系统版本、设备型号、系统信息
    • 设备连接信息，包括网络信息、信号强度
    • 设备状态信息，包括开关机数据、传感器数据
    • 设备摄像头信息，包括摄像头状态，发生错误后的摄像头画面信息
    • 设备及相关功能使用日志，包括系统和设备出现异常后的相关日志信息
产品功能使用和易用性分析
    • 交互数据，例如产品功能按钮点击次数，功能使用次数
    • 功能展示信息，例如屏幕页面停留时长
    • 配置选择信息，例如产品功能选项开启状态
    • 功能和产品使用信息，例如待机时长、打印时长、打印任务时间等相关信息
    2. 信息的使用
Creality承诺严格遵守国家相关法律法规。Creality将对收集的信息和内容严格保密，仅用于提高产品质量和改善产品服务的目的，不会提供给与该目的无关的第三方公司。如果您不提供这些信息，不会影响您的正常使用，但我们将无法解决您在使用过程中可能出现的可改善用户体验的问题。
关于我们如何使用您的信息，下面提供了详细示例：
    • 优化设备性能，例如分析打印中CPU占用率和内存使用情况
    • 分析故障内容改善售后服务，例如分析打印中错误信息和摄像头的错误识别历史信息，详细分析产品在使用过程中发生何种故障
    • 改善用户体验，例如分析用户机器功能的打开/关闭状态，判断此功能使用率以持续优化此功能改善体验
    3. 数据存储地点
Creality目前在中国和美国设有数据中心
    • 我们将在中华人民共和国大陆地区运营中所收集和产生的信息，在中国大陆地区的数据中心存储，但根据适用法律的规定允许跨境运输的除外。
    • 我们在非中国大陆地区运营中所收集和产生的信息，在美国的数据中心存储，但根据适用法律的规定允许跨境运输的除外。
    4. 数据存储期限
我们仅在实现数据收集目的所必需的服务期限内，或者遵守适用的相关法律要求保留您的个人信息。在此期间，您随时有权要求退出该计划或要求我们删除所向您采集来的数据。
    5. 您的权利
在您加入计划后，任何时候您需要查询、管理、删除我们所采集的您的数据信息，或您想要退出本计划以及您有任何有关我们所采集、处理数据的要求，您均可以通过以下方式联系我们：cs@creality.com ，我们将会及时给予处理。
