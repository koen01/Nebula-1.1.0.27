Advertencia de riesgo
    Rootear puede anular la garantía de ESTE PRODUCTO, y debe considerar cuidadosamente si necesita obtener permisos de root.

Riesgos de root
    Después de obtener acceso root, se abrirá el privilegio más alto del sistema de ESTE PRODUCTO, lo que permite acceder a todos los objetos del sistema operativo. El software general puede lograr más funciones utilizando el acceso root, pero el malware también debe utilizar el acceso root al invadir el sistema. Por lo tanto, considere cuidadosamente otorgar permisos de root a aplicaciones desconocidas.
    Después de obtener acceso root, los usuarios pueden modificar, eliminar, mover, copiar, etc. los archivos del sistema de ESTE PRODUCTO. Todas estas operaciones pueden causar excepciones en el sistema de ESTE PRODUCTO u otras aplicaciones.
    Después de obtener acceso root, los usuarios pueden eliminar programas preinstalados además de las aplicaciones de visualización, lo que incluye también varios componentes importantes del sistema. Eliminar estos programas importantes del sistema puede causar inestabilidad del sistema y un comportamiento anormal del programa.
    Después de obtener acceso root, si los usuarios otorgan permisos de root a algunos programas desconocidos, la información de privacidad del usuario en ESTE PRODUCTO puede filtrarse y resultar en un riesgo de seguridad para la información privada.
